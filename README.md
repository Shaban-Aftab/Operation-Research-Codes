# Operation Research LP Solver

Complete Linear Programming solver suite with Simplex, Big-M, and Sensitivity Analysis.

## Folders

- **LP_Solver_Final/** - Production-ready solver (recommended)
- **LP_Solver_helpers/** - Helper scripts and test files

## Quick Start

```bash
cd LP_Solver_Final
python integrated_solver.py
```

See individual folder READMEs for details.
